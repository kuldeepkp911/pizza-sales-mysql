# 🍕 Pizza Sales Analysis using MySQL

## 📌 Project Overview

This project analyzes a pizza sales dataset using **MySQL** to extract meaningful business insights.
The analysis covers basic, intermediate, and advanced SQL queries to understand sales performance, customer behavior, and revenue trends.

---

## 🎯 Objectives

* Analyze overall sales performance
* Identify top-performing pizzas
* Understand customer ordering patterns
* Evaluate revenue contribution across categories
* Track sales trends over time

---

## 🛠️ Tools & Technologies

* MySQL
* SQL (Joins, Aggregations, Window Functions)

---

## 📂 Dataset Description

The project uses the following datasets:

* **orders.csv** – Contains order date and time
* **order_details.csv** – Contains quantity and pizza IDs
* **pizzas.csv** – Contains pizza size and price
* **pizza_types.csv** – Contains pizza names and categories

---

## 🧠 Key Analysis Performed

### 🔹 Basic Analysis

* Total number of orders placed
* Total revenue generated
* Highest-priced pizza
* Most common pizza size
* Top 5 most ordered pizza types

### 🔹 Intermediate Analysis

* Total quantity of pizzas by category
* Distribution of orders by hour
* Category-wise pizza distribution
* Average pizzas ordered per day
* Top 3 pizzas based on revenue

### 🔹 Advanced Analysis

* Percentage contribution of each category to total revenue
* Cumulative revenue over time (Window Functions)
* Top 3 pizzas by revenue within each category

---

## 📊 Key Insights (Sample)

* Certain pizza categories contribute significantly more to total revenue
* Peak order hours help identify high-demand time periods
* A few pizza types dominate total sales and revenue
* Sales trends show consistent patterns over time

1. Import all CSV files into MySQL tables

4. Analyze outputs for insights
